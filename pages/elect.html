Ethernet cable tester

As part of my courses, I had an electronics project which involved designing an Ethernet cable tester. In this tester, we connect both RJ45 ends of the cable. Different sequences are coded to test if it functions properly.

This project taught me technical electronics modeling software to design the PCB. I learned soldering, and the code is written in C.
